## Installation
 Requirements
 - Have tmux installed
 - Have neovim installed

Install fots on the system
 https://sourcefoundry.org/hack/ (Enable it in the terminal)


Open tmux session
 ctrl + I (Capital i)

Install nodejs 
 npm install -g neovim

Install python
 /opt/homebrew/bin/pip3 install --user --upgrade neovim (On mac)

Open nvim
 :PlugInstall
